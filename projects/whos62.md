## Overview

whos62 is a lightweight trivia game where players guess a celebrity profile using age, height, and one rotating fact.

## Core Loop

- Get a clue set
- Make a guess
- See distance/accuracy hints
- Share your result streak

## Status

Prototype game loop and content schema in progress.
